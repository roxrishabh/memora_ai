# Memora AI

An Agentic Personal Knowledge System that indexes your local files and allows natural language search using LangGraph, RAG, Hybrid Search, and LLMs.

## Features

- Local-first
- Hybrid Search
- LangGraph Agent
- File Metadata Search
- OCR
- Code Search
- Semantic Retrieval

## ✅ Features

- Recursive filesystem scanner
- Incremental indexing
- SQLite metadata database
- SHA-256 change detection
- File type detection
- Programming language detection
- Python AST analysis
- Repository pattern
- Modular analyzer pipeline
